API Overview

LogiSense is a REST + WebSocket API that transforms raw delivery events into AI-powered intelligence. Base URL:

https://your-domain.com/api/v1

All responses follow this format:

{
  "success": true,
  "data": { ... }
}

Authentication

All authenticated endpoints require your API key in the X-API-Key header:

curl -H "X-API-Key: ls_your_api_key_here" \
  https://your-domain.com/api/v1/deliveries

Rate limits are returned in response headers:

HeaderDescription
X-RateLimit-LimitMonthly call limit for your tier
X-RateLimit-RemainingRemaining calls this month
X-RateLimit-TierYour current plan tier

Error Handling

{
  "error": "Validation Error",
  "message": "externalId is required"
}
StatusMeaning
400Bad Request — validation failed
401Unauthorized — invalid or missing API key
404Not Found
409Conflict — duplicate resource
429Rate Limit Exceeded
500Internal Server Error

Register Client

POST /api/v1/clients/register

Create a new client account and receive your API key. No authentication required.

Request Body

FieldTypeRequiredDescription
namestringrequiredYour name or org name
emailstringrequiredContact email
companystringoptionalCompany name
tierstringoptionalfree, starter, or growth
# Register
curl -X POST http://localhost:3000/api/v1/clients/register \
  -H "Content-Type: application/json" \
  -d '{"name":"Acme Corp","email":"dev@acme.com","tier":"free"}'

# Response
{
  "success": true,
  "data": {
    "id": "uuid",
    "apiKey": "ls_abc123...",
    "tier": "free"
  }
}

Deliveries

POST /api/v1/deliveries

Ingest a single delivery event. Creates or updates the delivery.

FieldTypeRequired
externalIdstringrequired
statusstringrequired
latitudenumberoptional
longitudenumberoptional
driverNamestringoptional
estimatedDeliveryTimeISO 8601optional

Valid statuses: picked_up, in_transit, out_for_delivery, delivered, failed, returned, delayed

POST /api/v1/deliveries/batch

Ingest up to 100 delivery events in a single call.

GET /api/v1/deliveries

List deliveries with pagination and filters.

GET /api/v1/deliveries/:id

Get delivery detail with full event timeline.

Insights

GET /api/v1/insights/:deliveryId

Get AI-powered risk scoring, anomaly detection, and ETA prediction for a delivery.

  "data": {
    "aiAnalysis": {
      "riskScore": 73,
      "riskLevel": "high",
      "anomalyFlags": ["Route deviation detected"],

    "anomalyDetection": {
      "hasAnomalies": true,
      "severity": "medium",
      "anomalies": [...]
    },
    "etaPrediction": {
      "estimatedMinutes": 45,

      "correctionMinutes": 30

GET /api/v1/insights/summary/batch

Get an executive-level batch summary with recommendations.

Usage

Health

WebSocket Tracking

Connect to the WebSocket endpoint for real-time delivery updates:

ws://your-domain.com/ws/tracking?apiKey=YOUR_API_KEY

Subscribe to a Delivery

ws.send(JSON.stringify({ 
  type: "subscribe", 
  deliveryId: "DELIVERY_UUID" 
}));
